## GIF assets for the extension

Put your locally converted animated GIFs in this folder.

Recommended filenames (used by the code):
- `logo.gif`
- `hero-idle.gif`
- `hero-hover.gif`
- `risk-medium.gif`
- `risk-high.gif`

If you use different names, update the paths in:
- `extension/entrypoints/popup/App.tsx`
- `extension/entrypoints/content.ts`
